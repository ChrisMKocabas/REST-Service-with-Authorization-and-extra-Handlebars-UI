Tech Stack: Node, Express, HTML, CSS, Handlebars Template Engine

A RESTful Backend service connected to a MongoDB Database to perform CRUD operations on New York Restaurants Registry data. 

It also uses handlebars.js to render web page version of the backend for a more user friendly alternative.

CRUD operations can be done via either exposed API endpoints or dynamically created and scalable web pages and forms pushing Handlebars template engine to its limits.

Web Service Handlebars interface is live to visually inspect on :

https://new-york-restaurants-registry.onrender.com
